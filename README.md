Chemobile
=========

## Getting Started

Clone this repo and run

```
flutter pub get
```

### Add this to your android/local.properties

```
flutter.minSdkVersion=21
flutter.targetSdkVersion=30
flutter.compileSdkVersion=30
```

### Start dev mode

```
flutter run
```

### Build apk

```
flutter build apk --release
```
